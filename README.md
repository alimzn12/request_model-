about learning.py:
Robust HTTP Client with Rate Limiting & Retry
📌 Overview

This mini project implements a production-style safe_get() function using Python's requests library.

The goal is to build a resilient HTTP client that:

Respects rate limits

Handles retries intelligently

Uses exponential backoff

Differentiates between retryable and non-retryable errors

Supports streaming downloads

Separates connection and read timeouts
